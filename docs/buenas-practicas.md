# Guía de buenas prácticas del proyecto

## Framework utilizado

El proyecto utiliza React con Vite para construir una landing page basada en componentes reutilizables, manteniendo una estructura clara, modular y fácil de mantener.

## Convenciones de nomenclatura

- Los componentes se nombran en PascalCase.
- Los archivos de componentes usan extensión `.jsx`.
- Las funciones y variables usan camelCase.
- Los nombres deben ser descriptivos y representar claramente su responsabilidad.

TestimonialsCarousel.jsx
