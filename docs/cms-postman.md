# Guía de uso del CMS con Postman

Este proyecto incorpora una API interna que funciona como un CMS simple para administrar contenido dinámico de la landing page del Centro de Negocios Santiago de SERCOTEC.

El CMS permite administrar:

- Servicios
- Testimonios
- Preguntas frecuentes
- Información de contacto recibida desde el formulario
